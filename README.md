# VoiceChatManager Bot

Hệ thống bot Discord quản lý kênh thoại tạm thời (Temporary Voice Channels) áp dụng mô hình Join-to-Create và Delete-when-Empty.

## Hạ tầng kỹ thuật
- **Ngôn ngữ:** Python
- **Thư viện:** discord.py, Flask, aiosqlite
- **Kiến trúc:** Cogs Framework
- **Cơ sở dữ liệu:** SQLite bất đồng bộ
